# 📦 TaskFocus: Полный комплект материалов для диплома

**Проект:** TaskFocus — интеллектуальный менеджер задач для пользователей с СДВГ  
**Технологии:** WPF, .NET, SQL Server, Entity Framework  
**Длительность:** 7-8 недель  
**Объём:** 60-80 страниц

---

## 📁 Список файлов

### 1. `БЫСТРЫЙ_СТАРТ.md` ⭐ НАЧНИ С ЭТОГО
**Что это:** Пошаговый план на первые 3 дня  
**Когда использовать:** Прямо сейчас, чтобы начать движение  
**Время:** 2-3 часа в день  
**Результат:** Окружение настроено, первые страницы написаны

---

### 2. `Диплом_План_работы_TaskFocus.md` 📋 ОСНОВНОЙ ДОКУМЕНТ
**Что это:** Полная структура диплома с пояснениями  
**Когда использовать:** На протяжении всей работы  
**Содержит:**
- Структуру всех 9 разделов диплома
- Что писать в каждом разделе
- Объём каждого раздела (в страницах)
- Научные источники для каждой темы
- Техническое описание архитектуры
- ER-диаграмму и UML
- План разработки по неделям
- Возможные вопросы на защите

**Размер:** ~300 строк детальных инструкций

---

### 3. `Шаблон_источников_диплом.md` 📚 БИБЛИОГРАФИЯ
**Что это:** Готовая таблица из 20 научных источников  
**Когда использовать:** При написании теоретической части  
**Содержит:**
- 20 проверенных источников по теме
- Ключевые тезисы каждой статьи
- Раздел диплома, куда вставить
- Готовые цитаты для вставки
- ГОСТ-оформление
- Карту использования по разделам

**Бонус:** Не нужно искать источники — всё уже найдено!

---

### 4. `Диаграммы_для_диплома.puml` 📊 ДИАГРАММЫ
**Что это:** Код диаграмм в формате PlantUML  
**Когда использовать:** При созданию иллюстраций  
**Содержит:**
- ER-диаграмма (сущности БД)
- Диаграмма компонентов (архитектура)
- UML-диаграмма классов (Domain Layer)
- Диаграмма последовательности (создание задачи)
- Диаграмма состояний (жизненный цикл задачи)
- Диаграмма деятельности (фильтрация по энергии)
- Диаграмма развёртывания

**Как использовать:**
1. Скопируй код на plantuml.com
2. Скачай PNG/SVG
3. Вставь в диплом

---

### 5. `Чеклист_по_неделям.md` ✅ ОТСЛЕЖИВАНИЕ ПРОГРЕССА
**Что это:** Детальный чек-лист на 7 недель  
**Когда использовать:** Каждое утро для планирования  
**Содержит:**
- День-by-дневные задачи
- Чек-боксы для отметки выполнения
- Критерии готовности каждой недели
- Итоговую статистику
- Советы по применению

**Как использовать:**
1. Распечатай или открой каждое утро
2. Отмечай выполненное ✅
3. Смотри на прогресс — мотивация растёт!

---

### 6. `User_Stories_и_Use_Cases.md` 🎯 ТРЕБОВАНИЯ
**Что это:** Описание функциональности с точки зрения пользователя  
**Когда использовать:** При проектировании и тестировании  
**Содержит:**
- 8 User Stories (формат: Как... Я хочу... Чтобы...)
- 5 Use Cases (подробные сценарии)
- Таблицу соответствия: User Story → Требование
- Приоритизацию MoSCoW (MVP vs v2.0)
- Сценарии для UX-тестирования
- Метрики успеха

**Бонус:** Можно показать руководителю — демонстрирует глубину проработки!

---

### 7. `README_ВСЕ_МАТЕРИАЛЫ.md` 📖 ЭТОТ ФАЙЛ
**Что это:** Навигация по всем материалам  
**Когда использовать:** Чтобы понять, что где находится

---

## 🗺️ ДОРОЖНАЯ КАРТА: Как использовать материалы

### Неделя 0 (Подготовка)
1. Прочитай `БЫСТРЫЙ_СТАРТ.md` — начни действовать
2. Открой `Шаблон_источников_диплом.md` — скачай статьи
3. Используй `Диплом_План_работы_TaskFocus.md` — пиши разделы 1.1, 2.1
4. Отмечай прогресс в `Чеклист_по_неделям.md`

### Неделя 1 (Проектирование)
1. Открой `Диаграммы_для_диплома.puml` — создай диаграммы
2. Используй `User_Stories_и_Use_Cases.md` — формализуй требования
3. Пиши раздел 4 в `Диплом_План_работы_TaskFocus.md`
4. Согласуй с руководителем

### Неделя 2-3 (Разработка)
1. Следуй техническому описанию в `Диплом_План_работы_TaskFocus.md`
2. Используй диаграммы из `Диаграммы_для_диплома.puml`
3. Реализуй User Stories из `User_Stories_и_Use_Cases.md`
4. Пиши раздел 5 (Реализация)

### Неделя 4 (Тестирование)
1. Используй сценарии из `User_Stories_и_Use_Cases.md`
2. Пиши раздел 6 (Тестирование)
3. Собери обратную связь

### Неделя 5-6 (Документирование)
1. Допиши все разделы по `Диплом_План_работы_TaskFocus.md`
2. Оформи список литературы по `Шаблон_источников_диплом.md`
3. Вставь диаграммы

### Неделя 7 (Защита)
1. Подготовь ответы на вопросы из `Диплом_План_работы_TaskFocus.md`
2. Сделай презентацию
3. Репетируй
