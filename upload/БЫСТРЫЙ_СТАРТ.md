## ✅ ДЕНЬ 1: Сегодня (2-3 часа)

### 1. Окружение (30 мин)
```
□ Установи Visual Studio 2022 Community (бесплатно)
□ Установи SQL Server Management Studio (SSMS)
□ Создай аккаунт на GitHub
□ Установи Git
```

**Ссылки:**
- VS: https://visualstudio.microsoft.com/ru/vs/community/
- SSMS: https://docs.microsoft.com/ru-ru/sql/ssms/download-sql-server-management-studio-ssms
- Git: https://git-scm.com/download/win

### 2. Репозиторий (15 мин)
```
□ Создай репозиторий на GitHub: "TaskFocus-Diploma"
□ Клонируй на компьютер
□ Создай папки: src/, tests/, docs/
□ Сделай первый commit: "Initial commit"
```

### 3. Литература (1 час)
```
□ Скачай 5 ключевых статей (список ниже)
□ Заполни таблицу источников (3-5 записей)
□ Прочитай введение 2-3 статей
```

**Приоритетные статьи:**
1. Asherson et al. (2012) — ADHD и time management
2. Ptáček et al. (2019) — time blindness
3. Li et al. (2026) — GenAI и ADHD (arXiv)
4. Perry et al. (2025) — neurodivergent productivity
5. Turjeman-Levi et al. (2024) — burnout и ADHD

### 4. Начни писать (30 мин)
```
□ Открой документ диплома
□ Напиши раздел 1.1 (Актуальность) — 1 страница
□ Используй цифры из статей
```

**Шаблон для актуальности:**
```
По данным ВОЗ, СДВГ диагностировано у 2.5-4% взрослого населения [1].
При этом исследования показывают, что 89% взрослых с СДВГ 
испытывают проблемы с executive function [2], что приводит к 
снижению продуктивности на 30-50% [3].

Существующие task-менеджеры (Todoist, Trello, Notion) разработаны 
для нейротипичных пользователей и не учитывают особенности 
восприятия времени и управления вниманием при СДВГ [4].

Таким образом, разработка специализированного инструмента, 
адаптированного под когнитивные особенности СДВГ, является 
актуальной задачей.
```

## ✅ ДЕНЬ 2: Завтра (3-4 часа)

### 1. Архитектура (1 час)
```
□ Нарисуй диаграмму слоёв (на бумаге или draw.io)
□ Определи 4 слоя: Domain, Application, Infrastructure, Presentation
□ Опиши, что в каждом слое
```

**Структура:**
```
┌─────────────────────────────┐
│  Presentation (WPF + UI)    │
├─────────────────────────────┤
│  Application (Services)     │
├─────────────────────────────┤
│  Domain (Entities, Rules)   │
├─────────────────────────────┤
│  Infrastructure (EF, DB)    │
└─────────────────────────────┘
```

### 2. База данных (1 час)
```
□ Нарисуй ER-диаграмму (3 сущности: User, Task, TaskHistory)
□ Опиши поля каждой сущности
□ Определи связи между таблицами
```

### 3. User Stories (30 мин)
```
□ Напиши 5 User Stories (шаблон: Как... Я хочу... Чтобы...)
□ Определи MVP (Must Have)
□ Определи, что в версии 2.0
```

**Пример:**
```
Как пользователь с СДВГ, 
я хочу видеть не более 3 активных задач, 
чтобы не чувствовать перегрузки.
```

### 4. Начни раздел 2 (1-2 часа)
```
□ Напиши раздел 2.1 (Обзор аналогов)
□ Опиши 3-5 инструментов: Todoist, Trello, Notion, Focusmate
□ Для каждого: что хорошо, что плохо для СДВГ
```

## ✅ ДЕНЬ 3: Послезавтра (4-5 часов)

### 1. Создай проект (1 час)
```
□ Создай Solution "TaskFocus" в VS
□ Добавь 4 проекта: Domain, Application, Infrastructure, Presentation
□ Настрой MaterialDesignInXAML (NuGet)
□ Настрой EF Core (NuGet)
□ Проверь, что всё собирается
```

### 2. Domain Layer (2 часа)
```
□ Создай сущность Task (поля: Id, Title, Description...)
□ Создай enum TaskStatus, Priority, EnergyLevel
□ Создай интерфейс ITaskRepository
□ Напиши класс TaskLimitPolicy (проверка лимита 3)
□ Напиши 2-3 юнит-теста для TaskLimitPolicy
```

### 3. Напиши ещё текста (1-2 часа)
```
□ Допиши раздел 2.2 (Нейробиология СДВГ)
□ Используй статьи Ptáček, Turjeman-Levi
□ Опиши: time blindness, executive dysfunction, dopamine
```

## 🎯 ЧТО ДЕЛАТЬ, ЕСЛИ ЗАСТРЯЛ

### Не знаю, с чего начать
→ Начни с актуальности (раздел 1.1). Просто опиши проблему своими словами.

### Не понимаю архитектуру
→ Нарисуй на бумаге: "У меня есть задачи, они хранятся в базе, я показываю их на экране". Это и есть архитектура.

### Страх, что не успею
→ Посчитай: 8 недель × 5 дней × 3 часа = 120 часов. Этого достаточно. Главное — регулярность.

### Не получается код
→ Переключись на текст. Напиши раздел 2 или 3. Вернешься к коду завтра с новыми силами.

### Устал
→ Сделай перерыв. Прогуляйся 15 минут. Выпей воды. Вернешься — будет легче.
