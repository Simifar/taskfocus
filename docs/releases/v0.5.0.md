# v0.5.0 - Russian-Only Product Version

## Summary

Релиз фиксирует продуктовую стратегию для дипломной версии: приложение остается русскоязычным без i18n-слоя.

## Changed

- Удалены неиспользуемые следы i18n.
- Приложение приведено к русскоязычной single-language модели.
- Нормализовано форматирование дат в пользовательском интерфейсе.
- Обновлен lockfile после удаления i18n-зависимостей.

## Fixed

- Исправлена смешанная строка `Detали` -> `Детали`.
- Исправлены lint-проблемы с синхронизацией state внутри effects.
- Убрана зависимость от системной локали в календарном `data-day`.

## Technical

- Решение отказаться от i18n принято как продуктовый компромисс для дипломной версии.
- Документация должна описывать приложение как русскоязычный продукт.

## Verification

- `npm.cmd run lint`
- `npm.cmd run build`
