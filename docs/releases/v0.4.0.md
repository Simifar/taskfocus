# v0.4.0 - UX and Profile Improvements

## Summary

Релиз улучшает визуальную систему, навигацию, карточки задач, auth UX и страницу профиля.

## Added

- Расширенная типографическая система.
- Brand color token.
- Переработанная страница профиля.
- Skeleton loading для профиля.
- Статистика пользователя в профиле.

## Changed

- Упрощена карточка задачи: действия перенесены в dropdown.
- Унифицированы метки навигации.
- Улучшена визуальная иерархия auth page.
- Добавлена ссылка восстановления пароля.

## Fixed

- Исправлены UI-проблемы с навигацией, архивом и week view.
- Улучшена согласованность шрифтов.

## Verification

- Ручная проверка основных dashboard views.
- Ручная проверка профиля.
- Ручная проверка auth page.
