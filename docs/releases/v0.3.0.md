# v0.3.0 - Reliability and API Hardening

## Summary

Релиз укрепляет backend, типизацию, обработку ошибок и надежность клиентских запросов.

## Added

- Structured logging.
- Next.js App Router error boundaries.
- Security headers: CSP, HSTS, frame protection.
- Optimistic updates для task mutations.

## Changed

- Строковые статусы и приоритеты заменены на Prisma enums.
- Константы уровней энергии и лимитов вынесены в server-side scheduling module.
- `apiFetch` получил timeout и корректную обработку `204 No Content`.

## Fixed

- Исправлены ошибки лимита активных задач.
- Исправлены edge cases API-ответов.

## Verification

- Ручная проверка task CRUD.
- Ручная проверка optimistic updates.
- Production build.
