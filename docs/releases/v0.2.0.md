# v0.2.0 - Authentication and Deployment Foundation

## Summary

Релиз добавляет основу авторизации и подготавливает приложение к деплою на Vercel + Neon.

## Added

- Email/password авторизация.
- Google OAuth через NextAuth.
- JWT cookie для custom auth flow.
- Prisma Adapter для OAuth-аккаунтов.
- Конфигурация для Vercel и Neon.

## Fixed

- Исправлены сценарии login/logout.
- Исправлена гонка состояния после входа.
- Уточнены настройки Google OAuth callback URL.

## Technical

- Добавлены server-side helpers для авторизации.
- Подготовлены переменные окружения для production.

## Verification

- Ручная проверка регистрации.
- Ручная проверка входа.
- Ручная проверка Google OAuth.
