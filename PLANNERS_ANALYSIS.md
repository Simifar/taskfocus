# 📊 Анализ структуры популярных планировщиков

## Todoist

### Левая панель (Sidebar)
```
Inbox          📥 (все быстрые задачи)
Today          📅 (только на сегодня)
Upcoming       📆 (предстоящие)
Filters & Labels
  - Для дома
  - Для работы
  - и т.д.
Projects       (папки/категории)
Trash
```

### Основная область
```
Заголовок с выбранным видом
╔════════════════════╗
║ Фильтр/Сортировка  ║
╠════════════════════╣
║  Задачи (список)   ║
╚════════════════════╝
```

### Ключевые особенности Todoist
- ✅ Чёткая разделение по приоритету (Inbox vs Today vs Upcoming)
- ✅ Projects (категории) в левой панели
- ✅ Быстрый поиск и фильтры
- ✅ Теги/метки для кросс-категоризации
- ✅ Рекурrentные задачи (repeat)
- ✅ Временные блоки (time estimates)

## TickTick

### Структура
```
На главной:
├── Inbox     (быстрые задачи)
├── Today     (на сегодня)
├── Upcoming  (на неделю/месяц)
├── Lists     (категории)
├── Calendar  (вид календаря)
├── Habits    (привычки)
└── Search
```

### Левая боковая панель (более детальная)
```
📌 Pinned Lists
  - Важные
  - Без даты

✅ My Lists (категории)
  - Работа
  - Личное
  - e-Shopping

📅 Calendar view

🎯 Goals
  - Monthly goals
  - Weekly goals
```

### Основной контент
```
┌─────────────────────────────────┐
│ [Вид: Список/Канбан/Календарь/]│
├─────────────────────────────────┤
│ Фильтр | Сортировка | Группировка
├─────────────────────────────────┤
│ [Задачи с временем выполнения]  │
└─────────────────────────────────┘
```

### Ключевые особенности TickTick
- ✅ Pinned Lists (избранные)
- ✅ Несколько видов (List/Kanban/Calendar/Gantt)
- ✅ Встроенный календарь
- ✅ Привычки (Habits)
- ✅ Временные оценки
- ✅ Подзадачи в модальном окне (детальный view)

## Microsoft To Do

### Простая структура
```
Левая панель:
├── Мой день        (сведённые задачи)
├── Важные          (помеченные)
├── Запланировано   (будущие)
└── Списки (My Lists)
   - Список 1
   - Список 2
```

### Основной вид
```
┌──────────────────────────────┐
│ Мой день (сегодня)           │
├──────────────────────────────┤
│ ✓ Завершённые (0)            │
│ ○ Активные                   │
│   - Задача 1                 │
│   - Задача 2                 │
│                              │
│ + Предложить список задач    │
└──────────────────────────────┘
```

### Ключевые особенности
- ✅ Минимализм (KISS - Keep It Simple)
- ✅ "Мой день" как основной фокус
- ✅ Предложения (Suggestions)
- ✅ Простая архитектура

## Notion

### Гибкая структура
```
Левая панель (Database):
├── All Tasks (всё)
├── My Tasks (мои)
├── Database views:
  - Calendar
  - Timeline
  - Gallery
  - Board (Kanban)
  - Table
└── Filters (любые комбинации)
```

### Основной контент
```
Выбранный вид (например, Board):
┌─────┬─────┬─────┐
│ To  │ In  │Done │
│ Do  │Prog │     │
├─────┼─────┼─────┤
│ Card│Card │Card │
│ Card│Card │Card │
└─────┴─────┴─────┘
```

### Ключевые особенности
- ✅ Множество видов (Views)
- ✅ Полная гибкость
- ✅ Базы данных (Relations)
- ✅ Фильтры + Сортировка + Группировка
- ⚠️ Сложная для новичков

## 🧠 Best Practices для СДВГ-приложения

### Что НУЖНО взять:
1. **Inbox** — для быстрого добавления без думания
2. **Today** — фокусировка на одном дне (СДВГ любит конкретность)
3. **Upcoming** — планирование без перегруза
4. **Categories/Lists** — разделение на контексты
5. **Quick View** — быстрый переключение между видами
6. **Calendar** — визуализация сроков (против time blindness!)
7. **Smart Suggestions** — "что мне делать сейчас?" (преодоление стены ужаса)
8. **Time Estimates** — понимание, сколько это займет (против прокрастинации)

### Что НЕ нужно (перегруз):
- ❌ Слишком много видов сразу
- ❌ Сложные фильтры по умолчанию
- ❌ Глубокая вложенность меню
- ❌ Много кнопок на экране

### Что НУЖНО добавить для СДВГ:
- ✅ **Energy Filter** (всё по энергии)
- ✅ **Max 3 active** (ограничение = спасение)
- ✅ **Soft deadlines** (диапазон дат, не точная дата)
- ✅ **Body doubling vibes** (мотивирующие quotes)
- ✅ **Quick wins** (видны маленькие задачи)
- ✅ **Time blocking** (время блокирования, не дата)

## 🎨 Summary: Структура для TaskFocus

**Главная идея:** Упростить настолько, чтобы СДВГ-пользователь не паниковал.

```
SIDEBAR (Левая)
├── Logo + User
├── Navigation
│   ├── Inbox (быстрая запись)
│   ├── Today (ГЛАВНЫЙ ФОКУС)
│   ├── This Week
│   └── [Category Lists]
├── Stats & Settings
└── Logout

MAIN (Правая)
├── Выбранный вид (Today/Inbox/Week/Calendar)
├── Energy Filter (если 3+ энергии → видны только подходящие)
├── Фокусная карточка (Energy status)
├── Список задач для этого вида
└── Мотивирующие квоты/прогресс
```
